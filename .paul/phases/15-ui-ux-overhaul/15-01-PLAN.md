<objective>
## Goal
Remove all notification and commenting features from the UI across admin, team, and portal surfaces. Database tables remain intact (hidden from UI only) for potential future restoration.

## Purpose
User requested simplification — notifications and comments are redundant features adding clutter. This aligns with the pivot to a personal-use tool where these collaboration features are unnecessary.

## Output
- Notification bell, notifications page, and all notification references removed
- Comment sections removed from task detail forms and portal dialogs
- Comment API route removed
- Dashboard notification sections removed
- Sidebar/mobile nav entries cleaned up
</objective>
