## Goal
Disable all team-related features from the UI. The app pivots to personal-use — admin is the only user. Team routes, sidebar items, invite system, task assignment UI, and "My Tasks" sections are removed.

## Purpose
User requested a pivot to personal-use tool. Team collaboration features (team member management, task assignments, team workspace) are no longer needed.

## Output
- Team Members nav item removed from admin sidebar
- /admin/team page and invite system removed
- All /team/* routes removed
- Invite registration pages removed
- Task assignment UI removed from task forms and lists
- "My Tasks" section removed from dashboard
- Assignee display removed from task cards, dialogs, and lists
</objective>
