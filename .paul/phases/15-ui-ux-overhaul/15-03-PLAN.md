# Plan 15-03: Streamline Project Navigation

**Phase:** 15 — UI/UX Overhaul & Simplification
**Plan:** 15-03
**Date:** 2026-04-20

## Problem

Navigating to a task requires 5 layers: Dashboard → Clients → Client Detail → Project Detail → Task Edit. The user finds this tedious and wants flatter navigation.

## Solution

1. **Add a Projects page** (`/admin/projects`) — flat list of all projects across all clients, with client name/color shown inline. Skips the Clients intermediary entirely.
2. **Update sidebar** — add Projects nav item between Dashboard and Tasks.
3. **Enhance dashboard** — add "Active Projects" section with quick-access cards that link directly to project task boards.
4. **Clean up leftovers from 15-02** — remove team member references from Quick Task Dialog, remove `task_assignments` join from project page query.

## Files to Create

- `app/admin/projects/page.tsx` — new Projects page listing all projects with client info, status, and direct links

## Files to Modify

- `app/admin/sidebar.tsx` — add Projects nav item with FolderKanban or Briefcase icon
- `app/admin/layout.tsx` — no changes needed (sidebar auto-includes new item)
- `app/admin/page.tsx` — add Active Projects section with quick-access cards
- `components/admin/dashboard-inner.tsx` — add DashboardActiveProjects component
- `components/admin/quick-task-dialog.tsx` — remove team member fetch, assignment UI (leftover from 15-02)
- `app/admin/clients/[clientId]/projects/[projectId]/page.tsx` — remove task_assignments from query and normalizedTasks mapping
- `app/admin/clients/[clientId]/projects/[projectId]/task-view-toggle.tsx` — no changes needed (TaskRow already clean)

## Acceptance Criteria

- [ ] AC-1: /admin/projects route shows all projects in a flat list with client color dot, client name, project name, status, and direct link to project tasks
- [ ] AC-2: Sidebar includes Projects nav item between Dashboard and Tasks
- [ ] AC-3: Dashboard shows Active Projects section with cards linking to project task boards
- [ ] AC-4: Quick Task Dialog no longer fetches team members or shows assignment dropdown

- [ ] AC-6: Build passes cleanly

## Navigation Flow After

- **Short path:** Dashboard → Projects → Project Tasks → Task Edit (3 steps)
- **Quick access:** Dashboard → Active Project Card → Project Tasks (2 steps)
- **Still available:** Clients → Client Detail → Project Detail (for client management)
