  <files>app/admin/clients/[clientId]/projects/[projectId]/actions.ts</files>
  <action>
    Create `app/admin/clients/[clientId]/projects/[projectId]/actions.ts` with 'use server':

    1. **createTaskAction(projectId: string, data: TaskFormData, assignedToTeamMemberId?: string | null)**:
       - Zod validation with same schema as form.
       - Insert into `tasks` table with project_id and all fields.
       - If assignedToTeamMemberId provided, insert into `task_assignments`.
       - If file was uploaded to temp path, use Supabase storage `copy()` to move from `${projectId}/temp/${uuid}/${filename}` to `${projectId}/${task.id}/${filename}`, then `remove()` the temp path.
       - `revalidatePath(`/admin/clients/*/projects/${projectId}`)`
       - Return { success: true, taskId: task.id }

    2. **updateTaskAction(taskId: string, data: TaskFormData)**:
       - Zod validation.
       - Update task in tasks table.
       - `revalidatePath(`/admin/clients/*/projects/${projectId}/tasks/${taskId}`)`
       - `revalidatePath(`/admin/clients/*/projects/${projectId}`)`

    3. **deleteTaskAction(taskId: string, projectId: string, designFilePath?: string)**:
       - If designFilePath exists, delete from storage: `supabase.storage.from('design-files').remove([designFilePath])`.
       - Delete task from tasks table (cascade handles task_assignments, comments).
       - `revalidatePath(`/admin/clients/*/projects/${projectId}`)`

    4. **updateTaskStatusAction(taskId: string, newStatus: 'todo' | 'in_progress' | 'done')**:
       - For kanban drag-and-drop.
       - `UPDATE tasks SET status = newStatus WHERE id = taskId`.
       - `revalidatePath` for the task list page.
       - Return success/error for rollback on failure.
  </action>
