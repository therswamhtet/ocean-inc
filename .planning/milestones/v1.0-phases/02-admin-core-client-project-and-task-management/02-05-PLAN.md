  <name>Task 1: Replace admin dashboard placeholder with metric cards</name>
  <files>app/admin/page.tsx</files>
  <action>
    Replace the current placeholder content in `app/admin/page.tsx` (RSC server component):

    1. Import `createClient` from `@/lib/supabase/server`, `format, startOfMonth` from `date-fns`.
    2. Run 4 parallel queries with Promise.all:

    ```typescript
    const today = format(new Date(), 'yyyy-MM-dd')
    const monthStart = format(startOfMonth(new Date()), 'yyyy-MM-dd')

    const [
      { count: activeProjects },
      { count: inProgress },
      { count: overdue },
      { count: completedThisMonth },
    ] = await Promise.all([
      supabase.from('projects').select('*', { count: 'exact', head: true }).eq('status', 'active'),
      supabase.from('tasks').select('*', { count: 'exact', head: true }).eq('status', 'in_progress'),
      supabase
        .from('tasks')
        .select('*', { count: 'exact', head: true })
        .lt('posting_date', today)
        .neq('status', 'done'),
      supabase
        .from('tasks')
        .select('*', { count: 'exact', head: true })
        .eq('status', 'done')
        .gte('updated_at', monthStart),
    ])
    ```

    3. Render 4 metric cards in `grid grid-cols-2 md:grid-cols-4 gap-4`:
    - Active Projects: `{activeProjects || 0}` — border border-border, rounded-sm, p-4
    - In Progress: `{inProgress || 0}` — same style
    - Overdue: `{overdue || 0}` — same style + `border-l-4 border-l-red-500`
    - Completed This Month: `{completedThisMonth || 0}` — same style
    - Each card: label (`text-sm text-muted-foreground`), value (`text-2xl font-bold mt-1`)

    4. Below metrics, fetch recent 5 notifications:
    ```typescript
    const { data: notifications } = await supabase
      .from('notifications')
      .select('*')
      .order('created_at', { ascending: false })
      .limit(5)
    ```

    5. Render notifications list with a heading "Recent Notifications":
    - Each notification as a `<form action={markAsRead}>` — click submits to mark as read.
    - Unread indicator: small `border-l-2 border-l-foreground` on left side.
    - Message text displayed, timestamp with `formatDistanceToNow`.
    - "View all notifications" link below the list → `/admin/notifications`.
