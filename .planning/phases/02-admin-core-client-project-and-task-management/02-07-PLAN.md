<task type="auto">
  <name>Task 2: Create design file download component with signed URLs</name>
  <files>components/admin/design-file-downloader.tsx</files>
  <action>
    Create `components/admin/design-file-downloader.tsx` ('use client'):

    1. Accept props: `filePath: string` (the storage path), `fileName: string` (display name).
    2. Download mechanism:
       ```typescript
       import { createClient } from '@/lib/supabase/client'

       async function download() {
         const supabase = createClient()
         const { data, error } = await supabase.storage
           .from('design-files')
           .createSignedUrl(filePath, 60)  // 60 second expiry

         if (error) return { error: error.message }
         window.open(data.signedUrl, '_blank')
       }
       ```
    3. Display: download button showing the file name as a link.
    4. On click: calls createSignedUrl, opens URL in new tab.
    5. Error state: show "Download link expired, please refresh" if URL expires.
  </action>
